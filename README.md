# OpenClaw Skills

我自己开发的 OpenClaw 技能集合。

## 技能列表

### checkin-manager
基于 SQLite 的打卡记录管理工具，支持分类校验、时间范围查询和汇总统计。用来追踪健身、学习、阅读等日常习惯。

### multimodal-vision
多模态图片理解与生成技能，集成 Gemini 3.1 Flash 模型，支持图片内容分析、OCR 提取和文本生成图片。

## 使用方式

每个技能文件夹包含独立的 `SKILL.md` 说明文档，按文档指引使用即可。

## 添加新技能

开发好的新技能复制到此目录，提交并推送到 GitHub。
